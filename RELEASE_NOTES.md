**2026.11.2** (BETA)

**Steam Overlay**

* Added a **Stack** slider. Instead of showing notifications one by one, they can now stack.
  This is useful when you receive more than one notification at the same time.
* Added a **Time to Removal** slider. By default, notifications disappear after 8 seconds. You can now choose a duration between 2 and 10 seconds.
* Added Group Invites.
* Added Friend Requests.
* Added World Invites.

**Notifications**

* Added profile images to notifications.
* Added group images to notifications.

**Profile**

* You can now see badges on your own profile.
* You can now edit, hide, or show your badges.

**People Tab**

* Added an **All Friends** tab to show all friends. I originally did not want to add this since you can already see all your friends in the sidebar, but people asked for it, so here it is.
* Changed the Favorites list view to a multi-grid layout.
* Changed the Blocked list view to a multi-grid layout.
* Changed the Muted list view to a multi-grid layout.
* Long status texts are now shortened to keep the layout responsive.

**World Tab**

* Changed the dropdown appearance.
* Added edit buttons to move worlds to different categories.
* Added a **Move to...** button to the right-click context menu.
* Added a context menu for edit mode to make quick changes.
* Added a separator to **All Favorites** to show categories.
* Added a success notification when making a change.

**Avatars Tab**

* Removed the **Favorite/Unfavorite** button from avatar cards. This is now handled through the context menu, giving avatar cards more space.
* Added a **Favorite** button to the context menu.
* Added edit buttons to move avatars to different categories.
* Added a **Move to...** button to the right-click context menu.
* Added a context menu for edit mode to make quick changes.
* Added a separator to **All Favorites** to show categories.
* Added a success notification when making a change.

**Fixes**

* Fixed an issue where the **Visitor** rank was not shown in the instance information or on the user profile when a Visitor-ranked player joined the instance.
* Fixed an issue that caused Group Invites to have missing images in the Timeline.
* Fixed an issue that caused instance time not to be saved.
* Fixed an issue that caused mismatches in time spent together.
* Fixed an issue that caused mismatches in world time spent.
* Rewrote the entire time spent system from the ground up so all three tracking types now share the same core system.
* Fixed an issue that displayed a low-resolution thumbnail of the world in the instance info modal.
