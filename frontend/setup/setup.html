Login to VRChat
First, you need to log in to VRChat in order to use all features of this app. Without logging in, you'll miss out on almost 90% of the functionality. This step is optional, you can skip it if you prefer.
folder_open
VRChat Launch Location
Please locate the launch.exe in your VRChat root folder. This is usually found in your Steam library folder. If you can't find it, right-click VRChat in Steam → Properties → Installed Files → Browse.
info Default: C:\Program Files (x86)\Steam\steamapps\common\VRChat\launch.exe
photo_library
VRChat Image Library
Point VRCNext to your VRChat screenshot folder so the app can monitor new photos, show them in your media library, and track which worlds they were taken in.
info Default: C:\Users\YourName\Pictures\VRChat
photo_camera
Camera Settings
Make sure to enable Metadata in your VRChat camera settings. This allows VRCNext to track which world each photo was taken in, so you can easily find that world again, even months later!
